# Real Upvote League Website

This is the flat version.

Everything is in the main folder so it works when uploading files through GitHub mobile.

How to upload:

1. Download this zip.
2. Unzip it.
3. Open the extracted folder.
4. Upload every file inside that folder to your GitHub repo.
5. Make sure index.html and styles.css are both visible in the main repo.
6. Import the repo into Vercel.
7. Leave build command blank.
8. Deploy.

Do not upload the zip itself.
